---

## 🗂️ Diagrammes

4 diagrammes sont disponibles représentant les contraintes et connections des pièces etc..
- Format : XML (fichier éditable avec [Diagrams.net](https://www.diagrams.net/), alias draw.io)
  

> Astuce : pour afficher une vignette dans ce README, exportez le diagramme en PNG et ajoutez l'image au dépôt (ex. `assets/diagramme-bdd.png`) puis insérez `![Diagramme BDD](assets/diagramme-bdd.png)`.

---


## 📁 Structure du dépôt

- `diagramme-BDD.drawio.xml` — source du diagramme (modifiable)
- `README.md` — ce fichier
- `assets/` — (optionnel) images exportées pour affichage dans le README
